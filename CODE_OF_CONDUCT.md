# Code of Conduct

This project expects direct, respectful collaboration.

## Expected behavior

- Treat maintainers, users, and contributors with respect.
- Discuss technical disagreement with evidence and concrete alternatives.
- Keep issues and pull requests focused on the project.
- Avoid harassment, personal attacks, discriminatory language, and repeated
  off-topic disruption.

## Enforcement

Maintainers may edit, hide, or remove comments and may restrict participation
for behavior that makes the project unsafe or unproductive.

If you need to report conduct concerns, contact the repository owner privately.
